Зачем

Функция импорта предназначена для того, чтобы из другого источника, в котором вы вели учет прежде, накопленные вами данные учета перенести в сервис «Мой удобный учет». Сделать это можно за несколько кликов.

Как выполнить импорт, пошаговый алгоритм
  1. Экспортируйте ваши данные учета из другого источника в таблицу Ексель. Таблица Ексель используется как промежуточный формат.
  2. Скопируйте и вставьте несколько строк учета из таблицы Ексель через буфер обмена в поле ввода для вставки данных.
  3. Нажмите кнопку «Импортировать». Увидите ошибки, если формат входных данных не соответствует ожидаемому.
  4. Исправьте формат входных данных в книге Ексель и повторите импорт нескольких строк. Корректируя формат входных данных добейтесь безошибочного импорта сначала нескольких строк.
  5. Импортируйте все ваши данные после того как отладите импорт на нескольких строках.
Требования к формату входных данных
  • Данные по каждой новой операции учета должны идти с новой строки.
  • Поля должны быть разделены символом табуляции.
  • Поля должны идти строго в перечисленном далее порядке.
  • Поля должны иметь строго указанный далее формат.
  • Количество полей в каждой строке должно быть ровно 6.
  • Поле 1 «Дата»: формат значения ДД.ММ.ГГГГ.
  • Поле 2 «Тип»: должно быть: «расход», «доход», «перевод». Регистр не важен.
  • Поле 3 «Сумма»: число, кроме цифр могут быть знаки «+ – . ,». Не цифры игнорируются.
  • Поле 4 «Счет»: строка. Если еще нет в системе, предварительно будет создан.
  • Поле 5 «Категория»: строка. Если еще нет в системе, предварительно будет создана.
  • Поле 6 «Комментарий»: строка, может быть пустой.
