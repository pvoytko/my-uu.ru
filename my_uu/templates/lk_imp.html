Зачем

Функция импорта предназначена для того, чтобы из другого источника, в котором вы вели учет прежде, накопленные вами данные учета перенести в сервис «Мой удобный учет». Сделать это можно за несколько кликов.

Как выполнить импорт, пошаговый алгоритм
  1. Экспортируйте ваши данные учета из другого источника в таблицу Эксель. Таблица Эксель используется как промежуточный формат.
  2. Скопируйте и вставьте несколько строк учета из таблицы Эксель через буфер обмена в поле ввода для вставки данных, расположенное выше.
  3. Нажмите кнопку «Импортировать». Увидите ошибки, если формат входных данных не соответствует ожидаемому.
  4. Исправьте формат входных данных в книге Эксель и повторите импорт нескольких строк. Корректируя формат входных данных добейтесь безошибочного импорта сначала нескольких строк.
  5. Импортируйте все ваши данные после того как отладите импорт на нескольких строках.
Требования к формату входных данных
  • Данные по каждой новой операции учета должны идти с новой строки.
  • Поля должны быть разделены символом табуляции (если копируете из Эксель, так и будет по-умолчанию).
  • Поля должны идти строго в перечисленном далее порядке.
  • Поля должны иметь строго указанный далее формат.
  • Количество полей в каждой строке должно быть ровно 6.
  • Поле 1 «Дата»: обязательное поле. Формат значения ДД.ММ.ГГГГ.
  • Поле 2 «Тип»: обязательное поле. Должно быть: «расход», «доход», «перевод». Регистр не важен.
  • Поле 3 «Сумма»: обязательное поле. Число. Кроме цифр могут быть знаки «+ – . ,». Иные знаки игнорируются.
  • Поле 4 «Счет»: обязательное поле. Строка. Если еще нет в системе, предварительно будет создан.
  • Поле 5 «Категория»: обязательное поле. Строка. Если еще нет в системе, предварительно будет создана.
  • Поле 6 «Комментарий»: может быть пустой. Строка.
